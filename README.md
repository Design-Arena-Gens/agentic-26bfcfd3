# Agentic Weekly Workout Planner

Agentic Weekly Workout Planner is a mobile-first Next.js experience for planning, customizing, and tracking a full week of training sessions. Map out strength, cardio, mobility, and recovery work, keep notes for each session, and monitor weekly volume and completion at a glance.

## Getting Started

Install dependencies and run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start planning your week.

## Features

- Weekly schedule with quick day switching and session counts
- Curated workout templates across strength, cardio, HIIT, mobility, core, and recovery
- Session customization (name, focus, intensity, duration, equipment, notes)
- Completion tracking with automatic weekly statistics
- Local storage persistence for personal planning

## Tech Stack

- Next.js App Router
- React with TypeScript
- CSS Modules for component-scoped styling
- Local Storage for client persistence

## Deployment

The project is ready for deployment on [Vercel](https://vercel.com/). Build with `npm run build` before deploying.
